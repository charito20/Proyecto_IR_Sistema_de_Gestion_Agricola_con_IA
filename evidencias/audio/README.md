# Audios

Esta carpeta contiene las grabaciones de audio obtenidas durante el proceso de levantamiento de requisitos del Sistema de Gestión Agrícola.

Aquí se almacenan las entrevistas realizadas a los participantes con su respectivo consentimiento informado.

Los archivos deben nombrarse con el siguiente formato:

fecha_participante_tipo

Ejemplo:
- 2026-06-20_entrevista_administrador_RaulVicenteMoreira.mp3
- 2026-06-21_entrevista_jornalero_AndreaAlejandroCedeno.mp3
